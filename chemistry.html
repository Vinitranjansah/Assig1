CHEMISTRY

Full Marks :70 Time : 180 minutes

Answer any five questions. All questions carry equal marks.

  1. Write the balanced chemical equation for the combustion of methane.
  2. Write the names and formulas of the first 10 alkanes.
  3. Write the definition and an example of a redox reaction.
  4. Write the Lewis structure and the molecular shape of carbon dioxide.
  5. Write the difference between a mixture and a compound.
  6. Write the periodic trends for atomic radius, ionization energy and electronegativity.
  7. Write the applications and drawbacks of nuclear energy.
