BASIC ELECTRICAL ENGNERRING

Full Marks :70 Time : 180 minutes

Answer any five questions. All questions carry equal marks.

  1. Write the definition and the formula of Ohm's law.
  2. Write the difference between AC and DC currents.
  3. Write the types and draw the symbols of common electrical components such as resistors, capacitors, inductors, etc.
  4. Write the methods and apply them to analyze simple circuits using Kirchhoff's laws, Thevenin's theorem, etc.
  5. Write the definition and the formula of power and energy in electrical systems.
  6. Write the types and draw the diagrams of common electrical machines such as transformers, motors, generators, etc.
  7. Write the applications and challenges of electrical engineering.
